# Mundo Pet - Fase 1

Este projeto foi desenvolvido para a disciplina de Fundamentos de Sistemas Web.

O site apresenta um petshop online chamado Mundo Pet, criado com HTML, onde é possível visualizar produtos e serviços oferecidos pela loja.

## Estrutura do site

O sistema é composto pelas seguintes páginas:

Home  
Página inicial com uma breve apresentação do petshop e links para as demais páginas do site.

Acessórios  
Apresenta produtos como brinquedos e camas para pets. Cada produto possui imagem, descrição e preço.

Rações  
Mostra opções de ração para cães e gatos, também com imagem, descrição e valor.

Higiene  
Exibe produtos de higiene para pets, como shampoo e tapete higiênico.

Serviços  
Apresenta os serviços oferecidos pelo petshop, como banho, banho e tosa e a opção de tele-busca.

## Tecnologias utilizadas

- HTML
- GitHub
- GitHub Pages


# Benjamin Pet Shop - Fase 2

## Sobre o Projeto

O Benjamin Pet Shop é uma plataforma web desenvolvida para facilitar o agendamento de serviços pet, oferecendo uma experiência prática, intuitiva e acessível para os clientes.

Esta fase representa a evolução do projeto, com melhorias na estrutura, interatividade e experiência do usuário.

## Descrição dos Ajustes Realizados

Para esta fase, foram realizados ajustes técnicos fundamentais para garantir a qualidade da entrega:

* **Estruturação de Arquivos**: O código foi organizado em pastas separadas (`/css`, `/js`, `/imagens`), seguindo boas práticas de desenvolvimento e facilitando a manutenção.
* **Refinamento do Layout**: Melhorias na responsividade do carrossel e dos cards de serviços utilizando Bootstrap 5.
* **Acessibilidade**: Inclusão de atributos `alt` em imagens, melhoria na semântica do HTML e aplicação de boas práticas para compatibilidade com leitores de tela.


## Novas Funcionalidades (Fase 2)

* **Carrossel Dinâmico**: Exibição interativa de banners com destaque para serviços e promoções, proporcionando melhor comunicação visual com o usuário.

* **Saudação Inteligente**: Função em JavaScript que identifica o horário do sistema e exibe uma mensagem personalizada (Bom dia, Boa tarde ou Boa noite) em um modal de boas-vindas.

* **Formulário de Agendamento Completo**:

  * Coleta estruturada de dados do cliente (CPF, telefone, e-mail, sexo);
  * Identificação do pet (nome, raça, idade);
  * Seleção de serviços (banho e tosa) e modalidade (tele-busca ou atendimento no local);
  * Validação de campos obrigatórios;
  * Uso de inputs específicos (`date` e `time`) para melhor experiência do usuário;
  * Interface organizada e intuitiva.


## Diferenciais do Projeto

* Interface amigável e responsiva;
* Uso de componentes modernos do Bootstrap;
* Interatividade com JavaScript baseada em contexto temporal;
* Organização do código seguindo boas práticas de desenvolvimento front-end.



## Tecnologias Utilizadas

* **HTML5 / CSS3**
* **Bootstrap 5** (layout e componentes)
* **JavaScript** (lógica temporal e manipulação do DOM)
