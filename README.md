# Mundo Pet

Este projeto foi desenvolvido para a disciplina de Fundamentos de Sistemas Web.

O site apresenta um petshop online chamado Mundo Pet, criado com HTML, onde é possível visualizar produtos e serviços oferecidos pela loja.

## Estrutura do site

O sistema é composto pelas seguintes páginas:

Home  
Página inicial com uma breve apresentação do petshop e links para as demais páginas do site.

Acessórios  
Apresenta produtos como brinquedos e camas para pets. Cada produto possui imagem, descrição e preço.

Rações  
Mostra opções de ração para cães e gatos, também com imagem, descrição e valor.

Higiene  
Exibe produtos de higiene para pets, como shampoo e tapete higiênico.

Serviços  
Apresenta os serviços oferecidos pelo petshop, como banho, banho e tosa e a opção de tele-busca.
